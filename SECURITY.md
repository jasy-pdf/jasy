# Security Policy

## Supported versions

jasy is in early alpha. Security fixes are made against the latest published `@alpha` release of each
package.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Report a vulnerability privately via the GitHub Security Advisories page:
[github.com/jasy-pdf/jasy/security/advisories/new](https://github.com/jasy-pdf/jasy/security/advisories/new).

All reports are verified and addressed promptly. Since jasy generates documents and e-invoices that may
carry sensitive data, please include the affected package, version, and a minimal reproduction.
