# Bundled fonts

Free, metric-compatible substitutes for the PDF standard-14 text families, embedded into ZUGFeRD
PDF/A-3 invoices (PDF/A requires every font embedded; the proprietary Helvetica/Times/Courier
can't be). Each maps a standard family, with all four styles (Regular/Bold/Italic/BoldItalic):

| Standard-14 | Bundled substitute (metric-compatible) |
| ----------- | -------------------------------------- |
| Helvetica   | Liberation Sans                        |
| Times       | Liberation Serif                       |
| Courier     | Liberation Mono                        |

(Symbol / ZapfDingbats are not bundled — they don't occur in invoices.)
