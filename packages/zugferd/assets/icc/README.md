# Bundled ICC profile

`sRGB.icc` — an **sRGB v2** ICC profile, embedded as the PDF/A `/OutputIntent` of ZUGFeRD invoices
(PDF/A must pin the output colour space; v2 is accepted by every PDF/A level).

**Origin / licence:** the `sRGB-v2-magic.icc` profile from
[saucecontrol/Compact-ICC-Profiles](https://github.com/saucecontrol/Compact-ICC-Profiles), which
dedicates its profiles to the **public domain (CC0 1.0)** — free to use, embed and redistribute.
The full CC0 1.0 legal text is in `LICENSE`. Kept verbatim (736 bytes), renamed to `sRGB.icc`.
